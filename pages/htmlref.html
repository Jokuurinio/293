Wie ist ein HTML Code aufgebaut?

Nachfolgend ist das HTML-Grundgerüst dargestellt:

<!DOCTYPE html> (Damit wird angegeben um was für ein Typ von Dokument es sich handelt)
        <html lang="en">
        <head>
            <meta charset="UTF-8">
            <meta name="viewport" content="width=device-width, initial-scale=1.0">
            <title>Document</title> (Was wird im geöffneten Tab oben angegeben)
        </head>
        <body> (Im Body wird die Struktur der Webseite gemacht)

        </body>
        </html>

Überschriften werden mit <h1> dargestellt

Paragrafen werden mit <p> dargestellt

Abschnitte werden mit Hilfe von <article> dargestellt

<!-- Mit diesen Zeichen wird ein Kommentar im HTML dargestellt -->

Verschiedene Farben und Schriftarten definieren

Farben und Schriftarten werden generell in einem eigenen Ordner und CSS file angelegt und abgespeichert.

            body{
                color : white;
                background-color: #696969;
            }
            nav{
                color: yellow;
            }
            p{
                color: green;
            }
            a{
                color: inherit
            }
